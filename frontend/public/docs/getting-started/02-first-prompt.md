# First Prompt

Создайте первый промпт за 5 минут.

## Шаг 1: Откройте Generator

Нажмите **Generator** в навигации или используйте горячую клавишу.

## Шаг 2: Опишите задачу

В поле "Task Description" введите что должен делать промпт:

```
Классифицировать отзывы клиентов как положительные, отрицательные или нейтральные
```

## Шаг 3: Выберите технику

- **Zero-Shot** — для простых задач
- **Few-Shot** — если нужны примеры
- **Chain-of-Thought** — для задач с рассуждением

## Шаг 4: Сгенерируйте

Нажмите **Generate** и дождитесь результата.

## Шаг 5: Сохраните в Library

Если промпт устраивает — нажмите **Save to Library**.
