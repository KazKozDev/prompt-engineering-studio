# Run Evaluation

Как протестировать промпт на датасете.

## Подготовка

1. Промпт сохранён в Library
2. Датасет загружен в Datasets

## Запуск оценки

### 1. Откройте Evaluation Lab

Перейдите в **Evaluation** в навигации.

### 2. Выберите промпт

В селекторе "Prompt" выберите промпт из библиотеки.

### 3. Выберите датасет

В селекторе "Dataset" выберите датасет для тестирования.

### 4. Настройте метрики

Доступные типы оценки:

| Тип | Описание |
|-----|----------|
| Quality | Точность, полнота, F1 |
| Robustness | Устойчивость к вариациям |
| Consistency | Стабильность ответов |

### 5. Запустите

Нажмите **Run Evaluation** и дождитесь результатов.

## Интерпретация результатов

- **Score > 0.8** — отличный результат
- **Score 0.6-0.8** — требует доработки
- **Score < 0.6** — нужна переработка промпта
