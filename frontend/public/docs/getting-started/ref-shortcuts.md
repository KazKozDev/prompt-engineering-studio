# Keyboard Shortcuts

Горячие клавиши для быстрой работы.

## Глобальные

| Клавиша | Действие |
|---------|----------|
| `⌘ K` | Поиск по документации |
| `⌘ S` | Сохранить текущий промпт |
| `⌘ Enter` | Запустить генерацию/оценку |

## Generator

| Клавиша | Действие |
|---------|----------|
| `⌘ Enter` | Сгенерировать промпт |
| `⌘ Shift S` | Сохранить в Library |
| `Tab` | Переход между полями |

## Evaluation Lab

| Клавиша | Действие |
|---------|----------|
| `⌘ Enter` | Запустить оценку |
| `Esc` | Остановить оценку |

## Prompt Library

| Клавиша | Действие |
|---------|----------|
| `⌘ N` | Новый промпт |
| `⌘ D` | Дублировать |
| `Delete` | Удалить выбранный |

## Навигация

| Клавиша | Действие |
|---------|----------|
| `⌘ 1` | Generator |
| `⌘ 2` | DSPy |
| `⌘ 3` | Datasets |
| `⌘ 4` | Evaluation |
| `⌘ 5` | Library |
| `⌘ ,` | Settings |
