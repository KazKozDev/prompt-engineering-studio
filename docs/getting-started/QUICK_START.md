# Quick Start - Get Going in 5 Minutes

Welcome! Let's get you creating optimized prompts right now.

## The 3-Minute Path

### 1. Go to Generator (30 seconds)

Click the **"Generator"** tab in the left sidebar.

You'll see:
- Text field for your prompt
- List of prompt engineering techniques
- Generate button

### 2. Write Your Prompt (30 seconds)

Type any task or question:

```
Examples:
- "Explain machine learning"
- "Write a thank you email"
- "Debug my Python code"
- "Summarize this article about AI"
```

Nothing fancy needed!

### 3. Select Techniques (60 seconds)

Pick 3-5 techniques to apply:

**Not sure which?** 
- Click "Reasoning" → Pick 3 techniques
- Or click "Select All" → Try everything

**Popular choices:**
- Chain-of-Thought (for explanations)
- Self-Critique (for accuracy)
- Tree of Thoughts (for creative tasks)

### 4. Generate (30 seconds)

Click **"Generate Enhanced Prompts"**

Wait 5-30 seconds...

### 5. You Got Results!

You'll see several versions of your prompt, each optimized differently.

**What to do:**
-  Read through them
-  Click Save on your favorite
-  Or copy to clipboard

## That's It!

You just generated optimized prompts. That's the core workflow!

## Common Questions (FAQs)

**Q: How long does generation take?**
- A: 5-30 seconds depending on provider and techniques

**Q: Which provider should I use?**
- A: Start with Gemini (fast) or Ollama (local/free)

**Q: Do I need an API key?**
- A: Only if using Gemini or OpenAI. Ollama is local, no key needed.

**Q: Can I save my prompts?**
- A: Yes! Click Save during generation, or save from results.

**Q: What if I don't like the results?**
- A: Try different techniques, modify your original prompt, try different provider

**Q: Can I edit the generated prompts?**
- A: Yes! Click the edit icon on any result.

**Q: Do you store my data?**
- A: No! Everything stays on your computer. Data never leaves unless you choose a cloud provider.

## Settings You Might Need

### To Use Gemini

1. Get API key: [makersuite.google.com](https://makersuite.google.com/)
2. Settings → API Keys → Add Gemini key
3. Done!

### To Use Ollama (Local)

1. Download: [ollama.ai](https://ollama.ai)
2. Run: `ollama serve` in terminal
3. That's it! (No API key needed)

### Change Theme

Settings → Theme → Pick Dark or Light

## Tips to Get Best Results

 **Be clear in your original prompt**
- Good: "Explain machine learning for a 10-year-old using simple analogies"
- Bad: "Explain ML"

 **Try multiple techniques**
- Different techniques work for different things
- Select 3-5 to compare

 **Test before using in production**
- Use Evaluation Lab to verify quality
- Don't just assume it's good

 **Save what works**
- Build up a library of prompts
- Reuse for similar tasks

 **Experiment**
- This is your lab!
- Try combinations you haven't seen
- You'll discover what works for YOU
