# Keyboard Shortcuts - Work Faster

Use keyboard shortcuts to speed up your workflow.

## Generation Shortcuts

| Shortcut | Action |
|---|---|
| **Cmd/Ctrl + Enter** | Generate prompts (after selecting techniques) |
| **Cmd/Ctrl + A** | Select all techniques |
| **Cmd/Ctrl + D** | Deselect all techniques |
| **Escape** | Close current dialog |

## Navigation Shortcuts

| Shortcut | Action |
|---|---|
| **Cmd/Ctrl + 1** | Go to Generator |
| **Cmd/Ctrl + 2** | Go to Datasets |
| **Cmd/Ctrl + 3** | Go to Evaluation Lab |
| **Cmd/Ctrl + 4** | Go to Prompt Library |
| **Cmd/Ctrl + 5** | Go to History |
| **Cmd/Ctrl + ?** | Help menu |

## Editing Shortcuts

| Shortcut | Action |
|---|---|
| **Cmd/Ctrl + C** | Copy selected text |
| **Cmd/Ctrl + V** | Paste |
| **Cmd/Ctrl + X** | Cut |
| **Cmd/Ctrl + Z** | Undo |
| **Cmd/Ctrl + Y** | Redo |
| **Cmd/Ctrl + A** | Select all text in field |

## Result Shortcuts

After generation:

| Shortcut | Action |
|---|---|
| **Cmd/Ctrl + C** | Copy result (when hovering over result) |
| **Cmd/Ctrl + S** | Save to library (when result focused) |
| **Tab** | Move to next result |
| **Shift + Tab** | Move to previous result |
| **Enter** | Expand result details |

## Search Shortcuts

| **Cmd/Ctrl + L** | Go to Library search |
| **Enter** | Go to first search result |

## Platform-Specific

### Mac

```
Cmd = 
Option = 
Shift = ⇧
Control = ^
```

Examples:
- Cmd+Enter = Generate
- Option+Cmd+C = Copy code block
- Shift+Cmd+R = Hard refresh

### Windows / Linux

```
Ctrl = Ctrl
Alt = Alt
Shift = Shift
```

Examples:
- Ctrl+Enter = Generate
- Ctrl+Shift+Delete = Clear cache
- Ctrl+Shift+R = Hard refresh

## Tips for Maximum Speed

### Workflow 1: Generate → Save

```
1. Select techniques (Cmd/Ctrl+A for all)
2. Type your prompt
3. Press Cmd/Ctrl+Enter to generate
4. Press Tab to navigate results
5. Press Cmd/Ctrl+S to save winner
```

**Total time:** 30 seconds for complete workflow

### Workflow 2: Quick Copy

```
1. Hover over result
2. Press Cmd/Ctrl+C (copies automatically)
3. Switch to other app
4. Paste
```

**Faster than clicking!**

### Workflow 3: Navigate Everywhere

```
1. Go Generator: Cmd/Ctrl+1
2. Go Library: Cmd/Ctrl+4
3. Find prompt: Cmd/Ctrl+F
4. Go back: Cmd/Ctrl+1
```

**No mouse needed!**

## Customizing Shortcuts

Currently shortcuts are fixed, but you can:

1. **Use browser extensions** to customize shortcuts
2. **Create bookmarklets** for common actions
3. **Set up keyboard mapping** on your OS

---

## Accessibility

If keyboard shortcuts don't work:

1. Check Caps Lock is off
2. Try in a different browser
3. Check if JavaScript is enabled
4. Try clicking instead (shortcuts optional)

---

## Pro Tips

 **Learn Cmd/Ctrl+Enter first** - Most used shortcut

 **Tab through results** - Faster than scrolling

 **Use Find (Cmd/Ctrl+F) for search** - Works everywhere

 **Hard refresh (Cmd/Ctrl+Shift+R) if stuck** - Clears cache

 **Press Escape to close dialogs** - Universal
